### 프로젝트
블록체인 분산원장에 거래가 기록되기 전에 불법 또는 이상 거래에 대한 사전탐지 및 알림을 제공하기 위한 블록체인 트랜잭션 Pool 실시간 모니터링 Agent 기술, 불법 거래로 인지 시 'Replace-by-Fee' 프로토콜을 활용한 거래 확정 전 차단기술로 이루어진 불법 거래탐지 및 이를 기반으로 한 거래 차단기술 개발
